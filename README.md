# LangC-curl-crient

## 実行手順

### 通常実行

- ビルド方法: Ctl+Shift+B  
- 実行方法: terminalを開き、`./main`と入力

### curlを使用した実行方法

- ビルド方法： `gcc <file name>.c -o <file name> -l curl`
- 実行方法： `./<file name>`

## curl, libcurlのインストール方法

- curlのインストール；`sudo apt install curl`
- libcurlのインストール：`sudo apt-get install libcurl4-openssl-dev`
